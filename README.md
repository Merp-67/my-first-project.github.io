# Мой путь в IT: с Amazon в frontend-разработку (2025–2026)

Это мой первый проект на чистом JavaScript — мотивационный трекер для junior frontend-разработчика.

## Что делает проект
- **Счётчик дней**: Показывает прогресс обучения (от старта до первой работы).
- **Статусы прогресса**: Кнопка меняет этапы от "работаю на складе" до "Зарплата 65k+ netto".
- **Калькулятор зарплаты**: Вводишь текущую/будущую зарплату — видит разницу в месяц/год. Данные сохраняются в localStorage.
- **Технологии**: HTML, CSS, JavaScript (DOM, события, функции, localStorage, Date, Math).
